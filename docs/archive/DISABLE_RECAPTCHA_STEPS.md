# How to Disable reCAPTCHA for Email/Password

## Current Location
You're on: **Authentication** → **Sign-in method** tab

## Where to Go
You need: **Authentication** → **Settings** tab

## Step-by-Step

1. **Click the "Settings" tab** (next to "Sign-in method" at the top)
2. **Scroll down** to find the "Fraud prevention" section
3. **Click on "reCAPTCHA"** (it should be highlighted/selectable)
4. **Look for options to disable reCAPTCHA for email/password**
   - There should be a toggle or setting
   - OR a "Manage reCAPTCHA" button that opens more options

## What You're Looking For

In the Settings → reCAPTCHA section, you should see:
- Options to enable/disable reCAPTCHA for different auth methods
- Email/password authentication setting
- Toggle or checkbox to disable it

## Alternative: If You See "Manage reCAPTCHA" Button

1. Click **"Manage reCAPTCHA"** button
2. This may open Google Cloud Console reCAPTCHA settings
3. Look for email/password authentication settings there
4. Disable reCAPTCHA for email/password

## After Disabling

1. **Save** the changes
2. **Wait 1-2 minutes** for changes to propagate
3. **Test login** - should work without timeout

The key is: **Settings tab**, not Sign-in method tab.
